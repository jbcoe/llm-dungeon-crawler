# Demo Alternative Content

An example of customising the game content (ie not the game engine) of `llm-dungeon-crawler`. The demo is a game called "Bathroom Apocalypse".

## Using the Alternative Demo Content

The files provided are drop-in replacements for the content stored in game/data subfolder of your game installation. Upon restarting the game in the usual way, there will be a whole new world, narrated in a slightly different tone !
